image底下有白边

首先 img 元素默认对齐方式为 vertical-align: baseline;，这就导致了，baseline 以下的部分被空了出来，显示了背景的白色。

问题找到了，对症下药可得出下面的解决方案：

  • 根本上消除 img 的对齐方式 —— 块状化：
  • 更改 img 对齐方式，以下三种均可
  • 更改行高，行高是两条 baseline 之间的距离，因此可以暴力的让行高消失
